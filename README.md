Team Sync UI
============

## Overview
Simple UI for communicating what a team is working on.  

## Environment Setup
* `npm install -g gulp jspm`

## Project Setup
* `git clone git@github.com:chriscantu/team-sync-ui.git`
* `cd team-sync-ui`
* `npm install`
* `jspm install -y`

## Running the project
* `gulp watch`
